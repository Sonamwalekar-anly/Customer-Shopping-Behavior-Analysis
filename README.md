
# 👨‍💻 Customer Behavior Analysis using Python, SQL & Power BI

An end-to-end Data Analytics project that analyzes customer shopping behavior to uncover valuable business insights. This project demonstrates the complete data analytics lifecycle, including data cleaning, exploratory data analysis, SQL querying, interactive dashboard creation, and business reporting.

This project is designed to showcase practical data analytics skills and can be used as a portfolio project for Data Analyst, Business Analyst, and Power BI Developer roles.

---

# 📌 Project Overview

The objective of this project is to analyze customer purchasing behavior and identify patterns that help businesses make data-driven decisions.

The project covers:

- 📊 Data Cleaning and Preprocessing using Python
- 🐍 Exploratory Data Analysis (EDA)
- 🗄️ SQL Database Creation and Analysis
- 📈 Interactive Dashboard Development in Power BI
- 📋 Business Insights and Recommendations

---

# 🎯 Objectives

- Clean and preprocess customer shopping data.
- Perform exploratory data analysis to identify trends.
- Load processed data into a SQL database.
- Write SQL queries to answer business questions.
- Build an interactive Power BI dashboard.
- Generate actionable business insights.
